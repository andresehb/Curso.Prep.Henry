Explicación de los siguientes conceptos

* Variables

Una variable es una caja que almacena un objeto
Para reconocerlo cuándo sea utilizado en algo
A esta caja se le da un nombre que indica lo que debería contener

* Strings

Son palabras y letras que se guardar en forma  de una frase
Son útiles para ser combinados y crear mensajes
También cada letra se le da un número que es útil para cuándo
Se necesite saber que tan larga es una frase

Funciones (argumentos, 'return')

* Funciones

Las funciones permiten resumir o dividir varias líneas de código
En un pequeño proceso de instrucciones
Que realizan una tarea o devuelven un resultado

* Argumentos

Un argumento es información que pasamos a una función
En la que el proceso la utilizará
Y la devolverá cuándo se invoque la función

* 'Return'

Finaliza con la tarea o instrucciones que
Le fueron ordenadas a una función
Para después devolver un valor

* Declaraciones 'if'

Es una estructura de control utilizada para tomar decisiones
Sirve para realizar una tarea u otra dependiendo de una regla
Revisa la información suministrada y si se cumple una regla
Se realizan las tareas o acciones que tienen que ser utilizadas
En el caso de que se cumpla
Si no se cumplen, se realizarán otras tareas hasta que la información
Cumpla con una condición
Si la información no cumple con ninguna condición se detendrá

* Valores booleanos ('true', 'false')

Los valores booleanos pueden ser simplemente como
Encendido o apagado, si o no, cierto o falso
Para ser utilizados hay que darles una 'pregunta'
Como por ejemplo: ¿10 es menor que 5? Lo que sería falso
